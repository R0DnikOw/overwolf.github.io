## November 20th 2023 - Tebex-Overwolf Subscriptions API

We have released the first bite of the new [Overwolf Subscriptions API](/start/monetize-your-app/subscriptions) documentation,
as well as the API itself.

With this, we hope to allow App developers an easier way to utilize subscriptions in their Apps,
leading to higher income, leading to better Apps.
