## Platform

- Updated the underlying [electron](https://npmjs.com/package/electron) version to [`22.3.13`](https://www.npmjs.com/package/electron/v/22.3.13).
- Performed various improvements and optimizations to the Overwolf Ads SDK.
- Added support for the Overwolf Electron Package Manager, a module targeted at supporting the different Overwolf API Modules.
  - The types repository complementing it can be found [here](https://www.npmjs.com/package/@overwolf/ow-electron-packages-types).
- Added support for the following Overwolf API Modules:
  - In-game Overlay - Module in charge of letting Apps utilize Overwolf's in-game Overlay.
  - Game Events Provider - Module in charge of letting Apps utilize Overwolf's [Live Game Data](/api/live-game-data) integration.
