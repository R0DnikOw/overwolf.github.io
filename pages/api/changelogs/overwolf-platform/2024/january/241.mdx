### API
- Added the optional `error` property to [`overwolf.streaming.onStopStreaming`](/api/streaming#onstopstreaming), for cases where the stream stopped successfully, but was stopped due to an error.
- Added the `uuid` property to [`overwolf.profile.onLoginStateChanged`](/api/profile#onloginstatechanged).
- Calling [`overwolf.extensions.current.setUserEmailHashes`](/api/extensions/current#setuseremailhasheshashes-callback) with an empty hashes object will now revert the App-specificed hashes.

### Bug Fixes
- The [`overwolf.streaming.onStreamingError`](/api/streaming#onstreamingerror) event will now be invoked whenever a stream is stopped due to an error, even if a video was still created.
- Fixed a bug where the `uuid` property would not be immediately ready in [`overwolf.profile.getCurrentUser()`](/api/profile#getcurrentusercallback) when changing accounts.
- Fixed a bug where the token generated by [`overwolf.profile.generateUserSessionToken()`](/api/profile#generateusersessiontokencallback) would be cached between client log-ins.
- Fixed a bug where the `isCursorVisible` property in [`overwolf.games.OverlayInfo](/api/games#overlayinfo-object) would sometimes be incorrectly set to true.
- Fixed a bug where trying to capture a screenshot in WoW would instead cause an `Unknown Error` to be thrown.
- Fixed a bug where the client would get stuck while loading large OPKs directly.
