### API
- New Method - [`overwolf.extensions.current.generateUserEmailHashes()`](/api/extensions/current#generateuseremailhashesemail-callback):
  - Generates and configures a user's Email hashes persistently directly from the raw email.
- New Method - [`overwolf.extensions.current.setUserEmailHashes()`](/api/extensions/current#generateuseremailhasheshashes-callback):
  - Configures a user's Email hashes persistently from pre-configured email hashes.

### Bug Fixes
- The [`overwolf.media.takeScreenshot()`](/api/media#takescreenshotcallback) method will no longer show the recording indicator
whenever it fails to execute.
- Fixed an issue when attempting to load a new App version while another App version is already installed.
