## `game_info`

### Info Updates

key          | Category    | Values                    | Notes                 | Since GEP Ver. |
------------ | ------------| ------------------------- | --------------------- | ------------- | 
scene | game_info | The current scene during the game  |  List of values See [notes](#scene-note)  |   238.0 |


#### *scene* note

Possible values:

* Menu - Main lobby menu
* Running - While in game
* Save - After save during a match

Data Example:

```json
{"info":{"game_info":{"scene":"Menu"}},"feature":"game_info"}
```

## `location`

### Info Updates

key          | Category    | Values                    | Notes                 | Since GEP Ver. |
------------ | ------------| ------------------------- | --------------------- | ------------- | 
location_camera | location | The current POV location of the camera  | See [notes](#location_camera-note)  |   241.0 |

main_map | location | The current main map name  | See [notes](#main_map-note)  |   241.0 |

#### *location_camera* note

Data Example:

```json
{"feature": "location", "category": "location", "key": "location_camera", "data": "[{\"x\":60.765,\"y\":-430.126,\"z\":24.668}]"}
```

#### *location_players* note

Data Example:

* This info-update includes the location of all the 4 possible players and will update only the characters that are moving 

```json
{"feature": "location", "category": "location", "key": "location_players", "data": "[{\"player_0\":{\"x\":-1086.69,\"y\":-224.49,\"z\":-0.595}},{\"player_1\":{\"x\":-1089.05,\"y\":-218.029,\"z\":-0.414}},{\"player_2\":{\"x\":-1077.75,\"y\":-229.75,\"z\":-0.507}},{\"player_3\":{\"x\":-1082.75,\"y\":-206.75,\"z\":0.168}}]"}
```

#### *main_map* note

Data Example:

```json
{"feature": "location", "category": "location", "key": "main_map", "data": "Wilderness"}
```

#### *sub_map* note

Data Example:

```json
{"feature": "location", "category": "location", "key": "sub_map", "data": "CAMPSITE"}
```
